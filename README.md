## Setup

### Server
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/21BCE7052-YourName-Game.git
   ```
2. Install dependencies:
   ```bash
   cd Pothuri_Saketh_21BCE7052
   npm install
   ```

### Run the Server
1. Start the server:
   ```bash
   node server.js
   ```
2. Open `http://localhost:3000` in your browser.
